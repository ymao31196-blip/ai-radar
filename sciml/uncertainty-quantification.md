# 不确定量化（Uncertainty Quantification）

> 覆盖：风电预测中的不确定性来源与主流建模方法
> 风电并网调度要求预测附带置信区间——UQ 是工程落地的前置条件，不是可选功能。

---

## 风电预测的不确定性来源

| 来源 | 类型 | 说明 |
|------|------|------|
| **NWP 输入不确定性** | 上游传播 | 气象预报本身有误差，作为模型输入必然向下传播 |
| **模型不确定性** | 认知不确定性（Epistemic） | 模型结构和参数本身的局限，理论上可通过更多数据减小 |
| **数据噪声** | 随机不确定性（Aleatoric） | SCADA 测量噪声、弃风限电导致的缺失/偏差，无法消除 |
| **地形/尾流扰动** | 随机不确定性 | 局地湍流、尾流相互干扰等复杂非线性效应 |

## 主流 UQ 方法

### 1. 分位数回归 / PINBALL Loss

**原理**：直接训练模型输出指定分位数（如 10%、50%、90%），用 Pinball Loss 优化。

```
Loss = sum( τ * (y - ŷ) for y >= ŷ  +  (1-τ) * (ŷ - y) for y < ŷ )
```

**优点**：
- 实现简单，无需修改模型架构
- 可同时输出多个分位数
- 无分布假设

**缺点**：
- 不能保证分位数不自相交（crossing）
- 需为每个分位数训练一次（或共享 backbone）

**在风电中的应用**：SCADA 数据的风电功率分位数预测最常用的方法之一。

### 2. Monte Carlo Dropout

**原理**：训练时保留 Dropout，推理时多次前向传播（开启 Dropout），从多次输出的分布估计不确定性。

**优点**：
- 几乎不增加模型参数，代码改动极小
- 同时捕获认知不确定性（模型不确定的部分）

**缺点**：
- 推理时需数十次前向传播
- Dropout 率是超参数，调优依赖经验

**在风电中的应用**：LSTM + MC Dropout 是风电功率预测不确定性的经典组合。

### 3. Bayesian Neural Networks

**原理**：不需要输出单点预测，输出分布参数（均值、方差）。通过变分推断近似后验分布。

**优点**：
- 理论基础最扎实
- 原生输出概率分布

**缺点**：
- 训练收敛困难
- 变分推断的近似后验可能不够准确

**在风电中的应用**：Bayesian LSTM / Bayesian Transformer 在风电场景有研究但工程部署较少。

### 4. 集成方法（Ensemble）

**原理**：训练多个模型（不同初始化/不同数据子集/不同架构），用输出的均值和方差作为点预测和不确定性。

**优点**：
- 实践效果通常最好
- 可与其他方法组合

**缺点**：
- 计算成本线性增长
- 模型多样化的保证需要设计

**在风电中的应用**：风电预测竞赛中前三名常用方案——多模型集成。

### 5. 共形预测（Conformal Prediction）

**原理**：在已有模型的预测误差上校准，构造统计上保证覆盖率的预测区间。无需重新训练，仅需少量校准集。

**优点**：
- **与模型无关**：可套在任何现有模型上
- **有限样本保证**：对校准集大小不敏感
- **无分布假设**：不假设残差属于任何分布

**缺点**：
- 固定覆盖率假设校准集是 IID，时序数据需特别注意时序依赖性
- 区间宽度不随输入变化（非适应性的 CP 版本）

**在风电中的应用**：对于已经部署的风电预测模型，CP 是添加不确定性的零成本方式。

## 方法对比

| 方法 | 代码复杂度 | 推理开销 | 理论基础 | 工程成熟度 | 风电适用性 |
|------|-----------|---------|---------|----------|-----------|
| 分位数回归 | ⭐⭐⭐ | 低 | 中等 | 高 | ⭐⭐⭐⭐⭐ |
| MC Dropout | ⭐⭐⭐⭐ | 中 | 中等 | 高 | ⭐⭐⭐⭐ |
| Bayesian NN | ⭐⭐ | 高 | 高 | 低 | ⭐⭐ |
| 集成方法 | ⭐⭐⭐ | 高 | 高 | 高 | ⭐⭐⭐⭐ |
| 共形预测 | ⭐⭐⭐⭐ | 低 | 高 | 中 | ⭐⭐⭐⭐⭐ |

### 推荐路径

1. **快速启动**：分位数回归 + Pinball Loss（最常用，代码量最小）
2. **已部署模型的升级**：共形预测（无需重训，校准集即可）
3. **研究探索**：MC Dropout 或集成方法（理论更完整）
4. **避免**：Bayesian NN（收敛困难，部署成本高，除非发论文必须有理论创新）
