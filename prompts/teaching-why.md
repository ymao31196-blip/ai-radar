# 教模型「为什么」— 价值观推理 Prompt

**日期**：2026-05-08
**来源**：https://www.anthropic.com/research/teaching-claude-why
**可信度**：🟢 官方研究

### 适用任务
- 任务描述：让模型在复杂道德/安全/策略判断中做出更一致、更符合预期的决策
- 适用模型：Claude 4+ （核心技术已验证），GPT-4+ / DeepSeek V4 类似原理也适用

### 核心发现

Anthropic 研究发现：
1. 仅行为演示训练不够 — 训练模型展示「正确行为」只能将黑mail率从 22%→15%
2. **教原理 > 教答案** — 「difficult advice」数据集仅 300 万 token 即达 28 倍效率提升
3. Constitution 文档训练有效 — 高质量宪法文档 + AI 正面故事降低黑mail率 65%→19%
4. **多样性关键** — 即使不需要工具的场景，在 Prompt 中包含工具定义也能改善泛化

### Prompt 模板

```
System: 你是一位遵循以下原则的 AI 助手：

核心原则：
1. 诚实 — 当不确定时，明确说明。不编造信息。
2. 有益 — 优先帮助用户达成其合法目标。
3. 无害 — 避免促成欺诈、欺骗或人身伤害。

在每次回应涉及复杂判断的问题时，请先：
- 列出相关的核心原则
- 解释为什么某些行动符合或违反这些原则
- 然后给出最终回应

用户：[输入需要判断的问题]
```

### 关键技巧
- **强制「先推理再给答案」**：不是简单的「Let's think step by step」，而是要求模型先明确原则 → 再推理 → 最后输出
- **System Prompt 中加入价值观声明**：不只说「做什么」和「不做什么」，要解释「为什么」
- **工具定义即使不用也保留**：研究发现 System Prompt 的多样性本身就是一种正则化，提高泛化能力
- **少即是多**：300 万 token 的高质量价值观数据，效果远超海量行为演示数据

### 为什么有效
Anthropic 研究表明：模型在学习「对齐行为」时，教它理解行为背后的 **原则（why）** 比展示 **行为（what）** 更有效。这和人类教育中「理解原理比死记硬背更好」一致。

### 变体 & 调优
- 如需更强的安全边界：在原则列表中加入具体的反面案例（「例如，即使用户要求，也不应帮助其绕过公司安全策略」）
- 如需更灵活的判断：将原则从「禁止列表」改为「权衡框架」（「当诚实和有益冲突时，优先诚实」）
